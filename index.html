Cómo funciona

De la acción a la NFT en cuatro pasos

Cada reto ambiental se completa con una foto que la IA valida antes de otorgar puntos y recompensas.

PASO 01

Elige un reto

Selecciona un desafío ambiental de la biblioteca, gratuito o premium.

PASO 02

Sube tu foto

Captura tu acción sostenible como evidencia directamente desde la app.

PASO 03

La IA evalúa

Nuestra IA analiza y valida que la evidencia cumpla con el reto.

PASO 04

Gana puntos y NFTs

Sumas puntos, desbloqueas una NFT y subes en el ranking.

La API de Astrogalaxy

Leer la documentación Obtener API key
astrogalaxy-api · evidencia 
# Subir evidencia y verificar con IA
curl -X POST https://api.astrogalaxy.eco/v1/evidence \
  -H "Authorization: Bearer $AG_KEY" \
  -F challenge=no-plastic-week \
  -F participant=student_5b_07 \
  -F photo=@evidencia.jpg
